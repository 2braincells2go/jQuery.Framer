Options

$('.framer').Framer({
	opacity: 0.8,
	overlayTime: 500,
	isOverlayClose: true,
	isAutoResize: true,
	isScroll: true,
	resizeRatio: 0.9,
	speed: 500,
	title: '<div id="edTitle"></div>',
	description: '<div id="edDescription"></div>',
	closeBtn: '<div class="close_btn"></div>',
	inner: $('<div id="Framer_inner"></div>'),
	width: 640,
	height: 360,
	iframe: '<iframe name="framer-iframe" frameborder="0"></iframe>',
	ajaxDataType: 'html'
});

オプション一覧

loadingColor
Activity Indicatorのカラー指定
overlayTime
Overlayのフェードスピード
isOverlayClose
Overlayにクリックで閉じる機能のON true または、OFF false
isAutoResize
自動でリサイズするかどうか
isAutoResize
自動でリサイズするかどうか
isScroll
スクロール時にFramerの位置も移動させるかどうか true or false
resizeRatio
リサイズの比率、ページサイズに対して
speed
フェードスピード
title
titleを表示したいときのHTMLコード指定 aのtitle属性の値を表示
description
descriptionを表示したいときのHTMLコード指定 aのdata-framer-description属性の値を表示
closeBtn
CloseボタンのHTML
inner
div#framer内にHTMLコード挿入したいときとか
width
デフォルトwidth
height
デフォルトheight
ajaxDataType
ajaxでロードするファイルタイプ
